
# Nexu Backend Challenge

API URL: https://nexu-backend-challenge-jarg-production.up.railway.app/

`npm run dev` start the server  
`npm run test` start the tests

### Endpoints

```
  GET    /api/brands
  GET    /api/brands/:id/models
  POST   /api/brands
  POST   /api/brands/:id/models
  PUT    /api/models/:id
  GET    /api/models
```
